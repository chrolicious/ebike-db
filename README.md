# E-Bike Database Nederland

Complete gids voor elektrische fietsen en e-crossers in Nederland.

## Nederlandse E-Crossers

Deze site bevat gedetailleerde informatie over 6 Nederlandse e-crosser modellen:

1. **Lectric XP Nederland** (€999) - Budget folding e-bike
2. **RadRunner Plus Nederland** (€1,699) - Familie cargo e-bike
3. **Super73 S2 Nederland** (€2,199) - Lifestyle e-bike
4. **Talaria Sting Nederland** (€4,899) - Premium crosser
5. **Zero FX Nederland** (€12,995) - A1 elektrische motor
6. **KTM Freeride E-XC Nederland** (€11,999) - Premium off-road crosser

## Features

- Nederlandse wetgeving compliance
- Affiliate links integratie
- Responsive design met Tailwind CSS
- SEO geoptimaliseerd
- Structured data markup

## Development

```bash
# Install dependencies

# Start development server
npm run dev

# Build for production
npm run build
```
